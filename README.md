# Student Management System (C)

## Description
This is a simple student management system written in C.  
It helps manage student information using file storage.

## Features
- Add student
- Display student list
- Search student by ID
- Save data to file
- Load data from file

## How to run

Compile:
gcc main.c student.c file.c -o main

Run:
./main

## Project structure
- main.c
- student.c / student.h
- file.c / file.h
- data.txt
